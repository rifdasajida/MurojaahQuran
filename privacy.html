Kebijakan Privasi Murajaah

Terakhir diperbarui: 12 April 2026

Murajaah adalah aplikasi gratis untuk membantu pengguna menghafal dan murajaah Al-Quran. Kami menghargai privasi kamu dan berkomitmen melindungi data pribadi kamu. Halaman ini menjelaskan informasi apa yang kami kumpulkan dan bagaimana kami menggunakannya.

1. Informasi yang kami kumpulkan

Saat kamu login menggunakan akun Google, kami mengumpulkan informasi berikut dari profil Google kamu:

  • Nama lengkap
  • Alamat email
  • Foto profil
  • ID akun Google (digunakan sebagai identifikasi unik di sistem kami)

Kami tidak meminta akses ke kontak, kalender, drive, atau data sensitif lainnya dari akun Google kamu.

2. Data yang kamu simpan di aplikasi

Selama menggunakan aplikasi, kamu dapat menyimpan:

  • Daftar surah yang sedang kamu hafal
  • Progress ayat yang sudah kamu hafal
  • Rekaman audio bacaan setoran kamu
  • Tanggal aktivitas hafalan (untuk fitur streak)

Semua data ini hanya bisa diakses oleh kamu sendiri melalui akun login kamu. Kami menggunakan teknologi Row Level Security untuk memastikan data kamu tidak dapat diakses oleh pengguna lain.

3. Mode tamu (tanpa login)

Jika kamu menggunakan aplikasi tanpa login, semua data kamu disimpan secara lokal di browser perangkat kamu. Kami tidak menerima atau menyimpan data apapun dari pengguna mode tamu di server kami.

4. Bagaimana kami menggunakan data

Data yang kami kumpulkan digunakan semata-mata untuk:

  • Mengidentifikasi akun kamu saat login
  • Menyimpan dan menampilkan progress hafalan kamu
  • Memungkinkan kamu mengakses rekaman setoran dari perangkat manapun

Kami tidak menggunakan data kamu untuk:

  • Iklan atau pemasaran
  • Analitik pihak ketiga
  • Dijual ke pihak manapun
  • Profiling atau pelacakan perilaku

5. Penyimpanan data

Data disimpan menggunakan layanan Supabase (database PostgreSQL dan storage). Rekaman audio disimpan di Supabase Storage dengan akses yang dibatasi hanya untuk pemilik akun.

6. Hak kamu

Kamu memiliki hak penuh atas data kamu:

  • Akses: Lihat semua data kamu kapan saja melalui aplikasi
  • Hapus: Hapus setoran individual atau seluruh akun kapan saja
  • Ekspor: Hubungi kami untuk request ekspor data kamu

Saat kamu menghapus akun, semua data terkait akan otomatis terhapus dari database kami secara permanen.

7. Keamanan

Kami menerapkan langkah-langkah keamanan teknis termasuk Row Level Security, HTTPS untuk semua komunikasi, dan otentikasi OAuth 2.0 melalui Google. Namun, tidak ada sistem yang 100% aman, jadi kamu juga perlu menjaga keamanan akun Google kamu sendiri.

8. Anak di bawah umur

Aplikasi ini dapat digunakan oleh semua umur. Untuk anak di bawah 13 tahun, kami menyarankan menggunakan aplikasi dengan pengawasan orang tua atau wali.

9. Perubahan kebijakan

Kebijakan privasi ini dapat diperbarui sewaktu-waktu. Perubahan akan diumumkan di halaman ini dengan tanggal pembaruan terbaru.
